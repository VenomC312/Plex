# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.3.x   | :white_check_mark: |
| 0.2.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

If you see a security flaw, please open a issue/pull request addressing it. Thanks!
